    应用前提

    “短线决策”指标是基于趋势交易思想研发的短线技术择时指标，当一只股票的长、中、短期的空方力量在同一时点力量衰竭，同时多方力量开始占据优势时，系统会提示参考信号，表示股价形成短期波段底部的概率较大。

    指标适用场景

    1.如果您希望在上升趋势的股票行情中建仓，又怕买在高点，那么短线决策指标会为您提供一个波段的相对低点作为上升趋势中的建仓参考位置。

    2.如果您希望在股票处于震荡行情时进行高抛低吸操作，短线决策指标同样会提供一个波段相对低点作为震荡行情中的操作参考。

    不适用的场景

    在股票处于下跌趋势时不建议使用短线决策指标。

    操作方法参考（数据在每个交易日收盘后15:30左右更新）

    1. 加仓方法：根据大数据的回测发现，短线决策指标在出现信号后的第二个交易日加仓最佳，若信号出现后的第二个交易日开盘时股价正向或负向波动超过2%，则应放弃此次交易机会。

    2. 止盈方法：加仓后3个交易日内（T+2日内），任何时间只要盈利超过2%即可考虑平仓止盈，若盈利未达到2%，也需在(T+2)日收盘前平仓。

    3. 止损方法：加仓后任何时间出现-2%浮亏时，建议止损离场。

    风险提示

    1. “短线决策”指标的信号仅供参考与借鉴，不构成买卖建议。

    2. “短线决策”指标的历史表现仅供参考与借鉴，不代表未来趋势。

    3. “短线决策”指标属于技术指标决策范畴，其未考虑宏观经济政策、股票所处行业以及个股基本面和消息面等因素，其应用的前提是在投资者自己已经选定股票、自己有建仓意愿，并且希望选择波段低点进行建仓的情景下使用。

    趋势量化系统之所以有效，是因为它能记录买方卖方的行为，股价产生波动的根本力量，是买卖双方的市场行为。任何消息本身，并不会造成股价的变动。即使有些消息确实使对某只股票的真实价值产生影响，也需要经过投资者的认可。

    怎样用ROIC、ROE、WACC进行公司估值

    投资回报率（ROIC）：是指投出和/或使用资金与相关回报（回报通常表现为获取的利息和/或分得利润）之比例。用于衡量投出资金的使用效果。
    净资产收益率（ROE）：反映股东权益的收益水平，用以衡量公司运用自有资本的效率。指标值越高，说明投资带来的收益越高。该指标体现了自有资本获得净收益的能力。
    加权平均成本（WACC）：是指企业以各种资本在企业全部资本中所占的比重为权数，对各种长期资金的资本成本加权平均计算出来的资本总成本。加权平均资本成本可用来确定具有平均风险投资项目所要求收益率。
    怎么用？
    (1) ROIC应大于WACC，若ROIC小于WACC说明公司投资回报率不如加权平均成本。则投资价值不高（一般公司的WACC在8%-12%）
    (2) ROE和ROIC都是越高越好，可作为长期投资价值的指标
