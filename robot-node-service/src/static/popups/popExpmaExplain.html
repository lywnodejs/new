研发理念

操盘线源于趋势理论和均线理论，秉持“大赚小赔“的操盘理念，并经过AI 量化技术的调整和改进而研发出的大赚小赔的操盘参考工具。它的主要作用，是帮助投资者克服人性弱点，智能提示股票中长线趋势性机会。

操作方法参考（数据在每个交易日收盘后15:30左右更新）

操盘线反应了个股当前市场下的中长线趋势状态，为投资者判断个股中长线趋势状态提供简单便捷的参考依据。
1.操盘线为红色时，代表中长线趋势看多，无论短线或中线操作，建议以加仓或持有为主；

2.操盘线为绿色时，代表中长线趋势看空，无论短线或中线操作，建议以减仓或空仓为主；

风险提示

1.操盘线无法避免系统性风险及个股突发风险

2.操盘线是基于历史价格数据的计算和分析得出的对个股所处中长线趋势的判断，仅可作为技术面参考，投资者据此操作风险自负。
